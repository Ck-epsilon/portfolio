API Starter v1.0.0 OAS 3.1

📦 技术栈

FastAPI · SQLAlchemy 2.0 · Pydantic v2 · JWT Auth · Async PostgreSQL

⚡ 性能

全异步 · 自动 OpenAPI 文档 · CORS 预配置 · Docker 即用

🔐 认证

JWT 登录/刷新/登出 · 密码哈希 · 令牌黑名单 · 角色权限

API 端点总览

GET/health 健康检查 — 返回服务状态和版本号
POST/auth/register 用户注册 — email + password → JWT tokens
POST/auth/login 用户登录 — 返回 access_token + refresh_token
POST/auth/refresh 刷新令牌 — 用 refresh_token 换新 access_token
POST/auth/logout 登出 — 将当前 token 加入黑名单
GET/users/me 获取当前用户信息（需认证）
PUT/users/me 更新当前用户资料（需认证）

快速体验

点击按钮模拟 API 调用
