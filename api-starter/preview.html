REST API Starter

Production‑ready scaffold with JWT auth, async database, and automatic API documentation.

Endpoints

GET/healthService health & version
POST/auth/registerCreate account → JWT tokens
POST/auth/loginEmail + password → tokens
POST/auth/refreshRenew access token
POST/auth/logoutBlacklist current token
GET/users/meGet profile (auth required)
PUT/users/meUpdate profile (auth required)

Built with

⚡

FastAPI

Full async. Automatic OpenAPI & Swagger UI. Type‑safe with Pydantic v2.

🗄️

SQLAlchemy 2.0

Async engine. PostgreSQL or SQLite. Alembic migrations ready.

🔐

JWT Auth

Register, login, refresh, logout. Password hashing. Token blacklisting.

📋

Pydantic v2

Request validation. Response serialization. Auto‑generated JSON Schema.

🌐

CORS Ready

Pre‑configured middleware. Works with any frontend out of the box.

🐳

Docker

Multi‑stage build. Health checks. Ready for deployment.
