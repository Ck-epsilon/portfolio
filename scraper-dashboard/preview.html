How It Works

📤

Create Task

POST a URL with CSS selectors and pagination config. The task enters an async FIFO queue.

🔄

Real-Time Stream

WebSocket pushes typed log events — navigation, extraction progress, page counts — as they happen.

🎯

Smart Extraction

JSON-defined field schemas drive Playwright's CSS selector engine. Structured or flat text modes.

📊

Export & Search

Download results as CSV. Search and sort the live results table in the dashboard.
